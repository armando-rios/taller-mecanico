# Sistem de Control de Taller Mecanico

Este es un sistema de control de taller mecanico

## Instrucciones de uso

1. Variables de entorno
2. Instalar las dependencias
3. Ejecutar el comando `npm run dev`

## Variables de entorno

- **FRONTEND**:

  1. `VITE_BACKEND_URL`: URL del backend

- **BACKEND**:

  1. `PORT`: Puerto del servidor
  2. `MONGO_URI`: URL de la base de datos
  3. `JWT_SECRET`: Clave secreta para la autenticacion

## Dependencias

Este proyecto utiliza turbopack como monorepo asi que necesitas tener instalado `pnpm`

## Comandos

- `pnpm install`: Instala las dependencias
- `pnpm dev`: Inicia el servidor de desarrollo
- `pnpm build`: Compila el proyecto
